At Everproof, we take the security and protection of your important files and personal information very seriously. Our system is architected with security being the most important design objective. As part of this commitment we undergo regular code reviews and penetration testing by independent third party auditors.

Security Process & Protocol

  • All communication is encrypted with SSL using a SHA256 certificate – this is the same level of protection used by e-commerce or online banking providers.
  • The Everproof application has been given an ‘A rating' according to SSLLabs Security Assessment.  

Data Backup

  • System information backed up using Microsoft Azure's Point In Time Restore - database can be restored to any point in time within a period of 7 days.
  • All records can be accessed and retained in the event of service disruption or discontinuation. 

Privacy & Personal Information

  • Everproof is compliant with privacy legislation (including the Australian Privacy Principles) and adheres to data protection protocols.
  • We have a published Privacy Policy that clearly outlines what data is collected and how it is used. This Policy undergoes periodic reviews and updates by Hall & Wilcox Lawyers.
  • Individuals retain complete ownership and control of data uploaded to Everproof. Data will NEVER be shared without consent.

Server Location & Hosting

  • Everproof’s physical infrastructure is hosted and managed by Microsoft. Microsoft has decades-long experience, its technology and security is state of the art and is used by businesses and individuals all over the world.
  • All data is stored in the Microsoft Azure East datacentre, located in Sydney.
